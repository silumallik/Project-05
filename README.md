# This is Digital – Website Recreation

This project is a modern front-end recreation of the **This is Digital** website, developed using a clean and well-structured codebase.

## Overview
The website focuses on smooth animations, interactive scrolling effects, and a visually engaging user experience.

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- GSAP (for animations)  
- Swiper.js (for sliders & carousels)  
- Feather / Smooth Scroll Effects  

## Key Highlights
- Smooth and scroll-based animations  
- Interactive sliders using Swiper.js  
- Modern UI with responsive layout  
- Optimized and structured code  

## Purpose
This project was created to practice advanced front-end concepts, animations, and smooth scrolling interactions using modern JavaScript libraries.
